Lenguajes aprendidos

HTML

Es un lenguaje de marcado usado para estructurar y presentar el contenido para la web. Las siglas de HTML significan Hyper Text Markup Language.

CSS

Es un lenguaje que permite dar estilo y presentar, de manera estructurada, un documento que fue escrito en lenguaje de marcado. Las siglas de CSS significan Cascading Style Sheets.

JavaScript

Es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web.
